# Hacking-Lab

Offensive security training from the ground up.  
100% Termux on Android. Zero shortcuts.

## Progress

| Week | Focus | Status |
|------|-------|--------|
| Week 1 | Networking, Linux internals, scripting | In progress |
| Week 2 | Web attacks, manual exploitation | Pending |
| Week 3-4 | Burp/Caido, SQLi, XSS, IDOR | Pending |
| Week 5-6 | AD, Kerberos, cloud misconfigs | Pending |

## Tools Built

- `host_discover.py` — TCP-based network host discovery
- `host_discover_v2.py` — Range-controlled scanner
- `probe_single.py` — Single-host service enumeration with banner grabbing
- `perm_decoder.py` — Linux permission decoder with special bits detection
- `suid_hunter.py` — SUID/SGID privilege escalation hunter
- `cred_hunter.sh` — Credential pattern hunter
- `recon_pipeline.sh` — Full recon automation (DNS, whois, headers, subdomains, nmap)

## Philosophy

Manual before automated. Understand before execute. Build before run.
